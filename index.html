David's foray into web development began with District Concerts, a site which aims to provide detailed information about all of the live music events which are taking place in Washington, DC. To build the site, David not only had to become far more familiar with DC's live music scene (a task which was particularly challenging since he lived in Belgium at the time), but he also needed to teach himself WordPress, become far more familiar with HTML and CSS, and develop a spreadsheet which could automatically generate conditional HTML. While building the site was not lucrative, the experience compelled him to complete a 6-month, full-stack web development program. Projects completed as part of that program include mE-Cards, which provides a simple way to create e-cards, Venue e-Valuator, which provides information about DC music venues, and Snapshot, which gives a quick overview of the day's pertinent information.
